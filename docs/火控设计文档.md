# 电动软弹枪火控系统设计文档

## 一、系统概述

基于 Arduino Nano 的软弹枪火控单元（FCU），支持 2S-6S LiPo 电池（开机自动检测电芯数），具备 OLED 屏幕菜单、预压延迟调节、多火控模式切换、简易弹量统计。

### 核心功能
- 每发保证推嘴/天梯桶归位（霍尔传感器闭环）
- 预压功能：归位后延时刹车，提前压缩弹簧，下一发零延迟响应
- 扳机松手不中断循环——一旦激发必须完成归位
- 4档快慢机：保险/单发/双行程/全自动
- OLED 屏幕 + 三按钮菜单系统
- 简易弹量计数

## 三、电路设计

### 3.1 电源拓扑

```
6S LiPo (22.2V nominal, 25.2V full) — also supports 2S-5S via auto-detect
  │
  ├──→ XT60 → 电机驱动回路 (大电流)
  │     │
  │     ├─ 电机+ ─┐
  │     │         ├─ 电机
  │     │         │
  │     │    ┌─ D(漏极) ← IRLB3034-MAIN (主驱动)
  │     │    │  G(栅极) ← Arduino D6 (PWM)
  │     │    │  S(源极) → GND
  │     │    │
  │     │    └─ IRLB3034-BRAKE (主动刹车)
  │     │       D(漏极) → 电机+ (跨接电机两端)
  │     │       G(栅极) ← Arduino D10
  │     │       S(源极) → GND
  │     │
  │     └─ 100μF/35V + 100nF (电机两端滤波)
  │
  └──→ Mini-360 降压模块
        │  输入: 2S-6S (7.4-25V)
        │  输出: 5V / 3A
        │
        ├─ Arduino Nano 5V (VIN 也可接，用 5V pin 更稳)
        ├─ OLED VCC
        ├─ 霍尔传感器×2 VCC
        └─ 分压电阻网络 (电池电压检测)
```

### 3.2 MOS 驱动与主动刹车

```
主驱动回路：
  Battery+ ───→ 电机+ ───→ 电机绕组 ───→ 电机- ───→ IRLB3034-MAIN(D) ───→ [S→GND]
                                                              │
                                                         G ← Arduino D6 (PWM=255 全开)

主动刹车回路（电机两端短路制动）：
  电机+ ───→ IRLB3034-BRAKE(D) ───→ [S→GND]
                  │
             G ← Arduino D10 (HIGH=刹车)

工作原理：
  正常驱动：MAIN=D6 HIGH, BRAKE=D10 LOW
  刹车制动：MAIN=D6 LOW,  BRAKE=D10 HIGH (延时100ms后全部LOW)
  注意：MAIN 和 BRAKE 绝对不能同时 HIGH（短路电池！）
```

### 3.3 传感器接线

```
897 霍尔开关模块（3线，已含上拉电阻和LED指示灯）：

  归位传感器：
    VCC → 5V
    GND → GND
    OUT → Arduino D3 (中断引脚 INT1)

  快慢机传感器 #1：
    VCC → 5V
    GND → GND
    OUT → Arduino D4

  快慢机传感器 #2（选配，4档需要2个）：
    VCC → 5V
    GND → GND
    OUT → Arduino D5

  897 模块特性：
    - 工作电压：4.5V~24V
    - 输出：开漏，有磁铁靠近时输出 LOW
    - 模块板上已有 LED 指示灯（磁铁靠近时亮）
    - 模块板上已有 10K 上拉电阻
```

### 3.4 控制输入接线

```
  扳机微动开关：
    一端 → Arduino D2 (内部上拉 INPUT_PULLUP)
    另一端 → GND
    （按下=导通到GND=LOW，松开=上拉HIGH）

  菜单按钮：
    UP     → Arduino D7 → GND（内部上拉）
    DOWN   → Arduino D8 → GND（内部上拉）
    SELECT → Arduino D9 → GND（内部上拉）
```

### 3.5 OLED 显示

```
  SSD1306 0.96" OLED (I2C)：
    VCC → 5V
    GND → GND
    SCL → Arduino A5 (PC5 / SCL)
    SDA → Arduino A4 (PC4 / SDA)
```

### 3.6 电池电压检测

```
  电池+ ──→ 100KΩ ──┬──→ Arduino A0 (ADC)
                     │
                   10KΩ
                     │
                   GND

  分压比：10K / (100K+10K) = 1/11
  6S 满电 25.2V → ADC 读值 = 25.2/11 × 1024/5 = ~469
  6S 存储 22.2V → ADC 读值 = 22.2/11 × 1024/5 = ~413
  6S 低压 19.8V → ADC 读值 = 19.8/11 × 1024/5 = ~369
```

## 五、软件架构

### 5.1 主状态机

```
                    ┌──────────┐
         扳机松开   │          │  扳机+可发射
       ┌───────────│  IDLE    │───────────┐
       │           │  (空闲)   │           │
       │           └──────────┘           │
       │                                  ▼
       │                            ┌──────────┐
       │                            │  FIRING  │  电机启动
       │                            │  (激发中) │
       │                            └─────┬────┘
       │                                  │ 归位传感器触发
       │                                  ▼
       │                         ┌──────────────┐
       │                         │ WAIT_PRE_COMP│  预压延时中
       │                         │ (预压等待)    │  (仅预压>0时)
       │                         └──────┬───────┘
       │                                │ 延时到
       │                                ▼
       │                         ┌──────────┐
       │                         │  BRAKING │  主动刹车
       │                         │  (刹车中) │
       │                         └─────┬────┘
       │                               │ 刹车完成
       │                               ▼
       │                         ┌──────────────┐
       │        (看模式+扳机)     │              │
       └─────────────────────────│  CYCLE_DONE  │
                                 │  (循环完成)   │
                                 └──────────────┘

关键设计原则：
  1. 一旦进入 FIRING 状态，扳机状态不再影响当前循环
     ——扳机松开也必须完成归位
  2. 跳回 IDLE 的唯一路径是 CYCLE_DONE 状态中
     确认扳机已松开（单发/双行程）或无连续需求（连发停）
  3. 连发模式：每个循环都完整归位，再根据扳机状态决定下一发
```

### 5.2 快慢机编码（2×897 霍尔，4档）

```
快慢机挡位        霍尔#2 (D5)  霍尔#1 (D4)
─────────────────────────────────────────
保险 (SAFE)         HIGH         HIGH       (两磁铁均不在传感器下)
单发 (SEMI)         HIGH         LOW        (磁铁仅在 #1 下)
双行程 (BURST)      LOW          HIGH       (磁铁仅在 #2 下)
连发 (AUTO)         LOW          LOW        (两磁铁均在传感器下)

注：897 霍尔有磁铁→OUT=LOW，无磁铁→OUT=HIGH（模块板上拉）
```

### 5.3 预压延迟原理

```
齿轮时序（简化）：
  
  电机转动 → 齿轮带动天梯桶后退 → 压缩弹簧 → 释放（发射）
           → 齿轮继续转动 → 回到归位点（霍尔触发）
  
  归位点磁铁经过霍尔传感器：
    ┌──┬──┬──┬──┐
    │  │  │██│  │  ← 齿轮上的磁铁（██=霍尔感应区）
    └──┴──┴──┴──┘
             ↑
          归位信号（霍尔输出 LOW）

  无预压（delay=0ms）：归位信号→立即刹车→齿轮停在归位点
  预压（delay=30ms）： 归位信号→等待30ms→刹车→齿轮多转了30ms
                       → 天梯桶开始被压缩 → 弹簧预压
```

### 5.4 菜单系统

```
主界面（默认显示）：
┌──────────────────────┐
│ ⚡24.5V  🔫SEMI      │  ← 第1行：电池电压 + 当前模式
│                      │
```

## 六、固件关键实现细节

### 6.1 传感器去抖

霍尔传感器在齿轮转动时可能产生抖动（磁铁边缘经过），需做软件去抖：

```cpp
// 归位传感器去抖——要确认稳定LOW才算归位
#define HOME_DEBOUNCE_US 200  // 归位需稳定 200μs
// 快慢机去抖——档位切换时可能有抖动
#define SELECTOR_DEBOUNCE_MS 50  // 档位稳定需 50ms
```

### 6.2 安全互锁

1. **MOS 互锁**：MAIN 和 BRAKE 绝对不能同时 HIGH（短路电池！），软件层双重保护
2. **欠压保护**：电池低于 3.3V/cell 时禁止发射（自动检测电芯数：2S=6.6V, 6S=19.8V），屏幕闪烁告警
3. **堵转保护**：电机连续运行超过 2 秒仍未归位 → 紧急停机，屏幕显示错误
4. **空匣保护**：弹量为 0 时禁止发射

### 6.3 中断 vs 轮询

归位传感器（D3）使用硬件中断（INT1）以保证不丢信号：
```cpp
// D3 = INT1，FALLING 边沿触发（霍尔有磁铁→LOW）
attachInterrupt(digitalPinToInterrupt(HOME_PIN), home_isr, FALLING);
```

## 七、装配注意事项

1. **大电流路径**：电池→电机→MOS→GND 这段用 14AWG 以上硅胶线
2. **信号线**：杜邦线即可，但远离电机线避免 EMI 干扰
3. **MOS 散热**：IRLB3034 在 50A 持续时约产生 2.5W 热量，TO-220 散热片足够
4. **磁铁安装**：钕磁铁（3mm 直径 × 2mm 厚）用环氧胶固定在齿轮上
5. **霍尔安装**：传感器面朝向齿轮，间隙 1-3mm，磁铁经过时模块上 LED 亮起即正常
6. **首次上电**：先用 3S 电池测试，确认所有功能正常后再上 6S
