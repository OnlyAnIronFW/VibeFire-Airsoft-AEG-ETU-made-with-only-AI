## 模块级接线表

### 1. 电源模块 (Mini-360 降压)
| Mini-360 | 连接目标 |
|----------|----------|
| IN+      | XT60 红 (电池+) |
| IN-      | XT60 黑 (电池-) |
| OUT+     | Arduino 5V pin + OLED VCC + 霍尔模块 VCC |
| OUT-     | Arduino GND + OLED GND + 霍尔模块 GND |

### 2. OLED 屏幕 (I2C, 4线)
| OLED | Arduino |
|------|---------|
| VCC  | 5V      |
| GND  | GND     |
| SCL  | A5      |
| SDA  | A4      |

### 3. 归位霍尔传感器 (897 模块, 3线)
| 897 模块 | 连接目标 |
|----------|----------|
| VCC (红) | 5V      |
| GND (黑) | GND     |
| OUT (黄) | Arduino D3 |

> 安装: 传感器面朝向齿轮磁铁, 间隙 1-3mm.
> 验证: 磁铁经过时模块上 LED 亮起.

### 4. 快慢机霍尔传感器 (897 模块 x2, 3线)
| 897 模块 A | 连接目标 |
|-----------|----------|
| VCC       | 5V       |
| GND       | GND      |
| OUT       | Arduino D4 |

| 897 模块 B | 连接目标 |
|-----------|----------|
| VCC       | 5V       |
| GND       | GND      |
| OUT       | Arduino D5 |

> 快慢机挡位编码 (磁铁靠近→LOW):
> - 保险: 两传感器均无磁铁 (D5=HIGH, D4=HIGH)
> - 单发: 磁铁仅靠近 #A (D5=HIGH, D4=LOW)
> - 双行程: 磁铁仅靠近 #B (D5=LOW, D4=HIGH)
> - 连发: 两传感器均有磁铁 (D5=LOW, D4=LOW)

### 5. 扳机微动开关
| 微动 | Arduino |
|------|---------|
| COM  | D2      |
| NO   | GND     |

> 按下=导通到GND=LOW, 松开=内部上拉HIGH

### 6. 菜单按钮 (3个)
| 按钮 | Arduino |
|------|---------|
| UP   | D7 → GND |
| DOWN | D8 → GND |
| SEL  | D9 → GND |

> 三个按钮一端接 Arduino 引脚, 另一端共地.

### 7. MOS 功率电路 (万能板上制作)

```
           主驱动 MOSFET                   刹车 MOSFET
    ┌─────────────────────┐      ┌─────────────────────┐
    │                     │      │                     │
电机- ─── D (漏极, 中间脚)  │      │ 电机+ ── D (漏极)     │
          │                │      │          │          │
          S (源极) ─── GND  │      │          S ─── GND  │
          │                │      │          │          │
          G (栅极) ── D6   │      │          G ── D10   │
          │                │      │                     │
    ┌─────┴─ 10KΩ ─── GND  │      └───── 10KΩ ─── GND   │
    │ (栅极下拉, 防浮空)    │       (栅极下拉)
    └─────────────────────┘

    IRLB3034 引脚定义 (TO-220, 面朝自己, 脚朝下):
     1 - G (栅极)    → Arduino D6/D10
     2 - D (漏极)    → 电机线
     3 - S (源极)    → GND (电池负极)
     tab(金属背板)    → D (漏极, 与 pin2 内部连通)

    关键: 主MOS的 D 接电机负极, S 接电池负极.
          刹车MOS的 D 接电机正极, S 接电池负极.
          绝对不能同时导通! 软件有死区保护.

    栅极下拉电阻 (10KΩ, G↔S): 防止Arduino未上电时MOS误导通.
```

### 8. 电池分压 (焊在万能板上)
```
电池+ ─── 100KΩ ─┬─── Arduino A0
                 │
                10KΩ
                 │
                GND

分压比: V_adc = V_bat × 10K/(100K+10K) = V_bat / 11
6S 满电 25.2V → A0 = 2.29V → ADC = 469
6S 存储 22.2V → A0 = 2.02V → ADC = 413
```

## 装配顺序

### 第一步: 低压测试 (用 3S 或 USB 供电)
1. 搭好降压模块→Arduino→OLED→按钮→霍尔
2. 烧录固件 (需安装 Adafruit_SSD1306 + Adafruit_GFX 库)
3. 确认 OLED 显示正常
4. 用磁铁靠近归位/快慢机霍尔，确认对应 GPIO 读到 LOW
5. 按下扳机确认触发逻辑
6. 进入菜单，调节预压/弹量/模式映射

### 第二步: 功率电路焊接
1. 在万能板上焊接两个 IRLB3034 + 10K 下拉电阻
2. 焊接电池分压电阻
3. 焊接电机滤波电容 (100μF 电解注意极性)
4. 用 14AWG 硅胶线连接大电流路径

### 第三步: 整机联调 (用 3S 电池)
1. 连接所有模块
2. 用 3S (11.1V) 电池供电
3. 空载测试 (电机未装齿轮):
   - 扳机→电机转→归位霍尔触发→刹车→停止 ✓
   - 松开扳机→不触发 ✓
   - 调预压→观察延时 ✓
   - 切换快慢机→各模式正常 ✓
4. 带载测试 (装齿轮):
   - 确认归位准确、无过冲
   - 调优预压延迟值
   - 连发测试循环稳定性

### 第四步: 上 6S
1. 确认 3S 测试全部通过
2. 换上 6S 电池
3. 检查 MOS 发热 (正常温升 <40°C)
4. 全自动连发压力测试

## 常见问题排查

| 现象 | 可能原因 | 检查点 |
|------|---------|--------|
| OLED 不亮 | I2C 地址不对 | 用 I2C Scanner 扫描, 常见 0x3C/0x3D |
| 归位不停 | 霍尔未触发/安装太远 | 传感器 LED 是否在磁铁经过时亮 |
| 电机不转 | MOS 坏/接线错误 | 用万用表测 MOSFET 栅极电压 (>4V) |
| 刹车无效 | 刹车MOS烧毁/接线错误 | 刹车时电机两端应为0V |
| MOS 发烫 | 栅极驱动不足/散热不够 | 检查 10K 下拉, 加散热片 |
| 串口乱码 | 波特率不匹配 | 确认 115200 |
| 堵转误报 | STALL_TIMEOUT_MS 太短 | 尝试改为 3000ms |

## 可选增强

- **蓝牙调参**: 加 HC-06 蓝牙模块 (TX→D0, RX→D1), 手机串口 APP 调参
- **陀螺仪稳控**: 加 MPU6050 模块, 检测枪身姿态控制发射
- **弹匣计数重置**: 弹匣插入检测微动 → 自动重置弹量
- **电池单体检测**: 外接 6S 平衡头→分压网络, 每节独立监控
